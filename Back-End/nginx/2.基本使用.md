# Nginx基本使用

一般安装好nginx之后，系统会自动注册好`nginx`命令（如：/usr/local/bin/nginx），这样就可以直接在命令行使用。

- `nginx`: 启动nginx
- `nginx -s stop`: 关闭nginx
- `nginx -s reload`: 重启nginx，一般在改完配置之后都需要重启

http #http块
{
    ... # http全局块

    server # server块
    { 
        ... # server全局块
        location [PATTERN] # location块
        {
            ...
        }
        location [PATTERN] 
        {
            ...
        }
     }
    server
    {
        ...
    }
    ... # http全局块
}
